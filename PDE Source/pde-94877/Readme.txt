Requirements:
    * .Net Framework 4.0
    * Visual Studio 2010
    * Visual Studio 2010 SDK
    * Visual Studio 2010 Visualization & Modeling SDK
    * T4 Toolbox (http://t4toolbox.codeplex.com)

1. open and start Tum.PDE.LanguageDSL. Tum.PDE.DomainModelDSL will be opened in the experimental hive.
2. compile Tum.PDE.ToolFramework
3. lngdsl file can be opened in the graphical editor. (open with --> wpf editor)
4. transform templates
5. start


Setup: /Setup/... 


Hint: Should you be getting the "Exception has been thrown by the target of an invocation" error, pls delete the Language DSL Plugin by removing the following folder:
"..\AppData\Local\Microsoft\VisualStudio\10.0Exp\Extensions\Tum\Process Development Environment Language"
